Security

Multi-Layer Protection

Security at every step — only real voters' votes are counted

🖐

Fingerprint Biometrics

ISO-format template stored via SecuGen HU20 scanner. Fingerprint match required for both login and vote.

🔒

Bcrypt Password Hashing

Passwords are never stored in plain text — hashed with bcrypt salt rounds before entering database.

⏰

Voting Schedule Lock

Voting only possible within the time window set by admin. Early or late votes are blocked server-side.

🚫

Double Vote Prevention

Each voter can vote only once. Server-side verification ensures no client-side tampering.

📍

Location Tracking

GPS coordinates captured at time of vote. City/area name is retrieved via OpenStreetMap Nominatim.

🛡️

Session Management

Separate admin and voter sessions. Every protected route verifies session — unauthorized access blocked.
