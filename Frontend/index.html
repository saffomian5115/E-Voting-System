Security

Multi-Layer Protection

Har step pe security — sirf asli voter ka vote count hota hai

🖐

Fingerprint Biometrics

SecuGen HU20 scanner se ISO-format template store hota hai. Login aur vote dono pe fingerprint match zaroori hai.

🔒

Bcrypt Password Hashing

Password kabhi plain text mein store nahi hota — bcrypt salt rounds ke saath hash hoke DB mein jaata hai.

⏰

Voting Schedule Lock

Admin jo time window set kare, sirf usi ke andar vote possible hai. Early ya late vote server side block hoti hai.

🚫

Double Vote Prevention

Ek voter sirf ek baar vote de sakta hai. Server side check — client trust nahi kiya jaata.

📍

Location Tracking

Vote ke waqt GPS coordinates capture hote hain. OpenStreetMap Nominatim se city/area naam nikala jaata hai.

🛡️

Session Management

Admin aur voter ka session alag. Har protected route pe session verify hoti hai — unauthorized access block.
